使用方法

 1. 登陆 域名+manage 进入总后台， 账号是，admin 密码是 lizhili 
 2. 创建项目 和人员 并分配权限 
 3.  登陆网站主页
 4. 登陆，不同权限拥有不同的功能
 5. 后台可以自己管理自己的账号。
